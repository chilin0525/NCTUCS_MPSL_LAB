### 3.2 Photoresistor and ADC

說說解析度 (Resolution) 是甚麽？改變解析度對觀察到的取樣值有何影響？

每次訊號轉換結束的數值範圍大小，值越大表示 resolution 越細，  因此每次取樣的就越細資料取的也將更完整更細一些。

(Coding) 將解析度更改為 8-bit
