# P3

## GPIO

### Buzzer:

IN <==> PA5(TIM2 Channel)

VCC <==> 3V3

GND <==> GND

### Keypad

pb3-6 : input

X0 == pb6 == pin4 == col1

X1 == pb5 == pin3 == col2

X2 == pb4 == pin2 == col3

X3 == pb3 == pin1 == col4
    
pc0-3 : output to test

Y0 == pc0 == pin8 == row1

Y1 == pc1 == pin7 == row2

Y2 == pc2 == pin6 == row3

Y3 == pc3 == pin5 == row4
